- **React.js 18** — UI framework
- **Framer Motion** — Animations
- **React Icons** — Icon library
- **CSS Variables** — Design tokens & theming
- **Google Fonts** — Syne, Space Mono, DM Sans

## 🛠️ Local Development

```bash

npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🌐 Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm install -g vercel
vercel
```

### Option 2 — GitHub + Vercel Dashboard
1. Push this project to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **"Add New Project"**
4. Import your GitHub repository
5. Vercel auto-detects Create React App — click **Deploy**
6. Your portfolio is live! 🎉

> The `vercel.json` file handles SPA routing automatically.
